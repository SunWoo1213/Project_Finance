# Utils 개발 방향성

이 폴더는 순수 함수와 정적 매핑을 둡니다.

## 현재 책임

- `constants.js`: ticker와 표시 이름 매핑
- `formatters.js`: 가격, 변화율, 시가총액 등 표시 포맷
- `validationSchemas.js`: 폼 검증 스키마

## 개발 원칙

이 폴더의 코드는 가능한 한 순수해야 합니다. React state, DOM, API 호출, localStorage 접근을 넣지 않습니다.

새 자산을 추가할 때는 `ASSET_NAMES`만 수정하고 끝내지 말고 백엔드의 자산 목록과 히스토리 API 처리도 함께 확인합니다.

포맷터는 화면별 임시 문자열 처리보다 우선 사용합니다. 가격 단위, 채권 수익률, 원화/달러 표시 규칙이 화면마다 달라지지 않도록 이곳에 모읍니다.
